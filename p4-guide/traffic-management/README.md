Some references on Approximate Fair Drop (AFD), first described in
this paper:

+ "Approximate Fairness through Differential Dropping", Rong Pan, Lee
  Breslau, Balaji Prabhakar, and Scott Shenker, ACM Computer
  Communications Review, Volume 33, Number 2, April 2003,
  https://people.eecs.berkeley.edu/~istoica/classes/cs268/10/papers/afd.pdf

Also mentioned in many other papers, of which the below are a sample
of two that I found and selected somewhat arbitrarily.

+ "AF-QCN: Approximate Fairness with Quantized Congestion Notification
  for Multi-tenanted Data Centers", Abdul Kabbani, Mohammad Alizadeh,
  Masato Yasuda, Rong Pan, and Balaji Prabhakar
  https://people.csail.mit.edu/alizadeh/papers/afqcn-hoti10.pdf

+ "Controlling software router resource sharing by fair packet
  dropping", Vamsi Addanki, Leonardo Linguaglossa, James Roberts, and
  Dario Rossi,
  https://perso.telecom-paristech.fr/drossi/paper/rossi18networking.pdf

This link appears to be a Ph.D. student's implementation of AFD
intended for exeution in the Linux kernel:

+ https://www.cct.lsu.edu/~xuelin/afcodel/sch_afq.c
